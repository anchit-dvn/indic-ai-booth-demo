# Place pre-recorded audio samples here for the Call Center demo.

## Required Files

Create the following WAV files (16kHz, mono recommended):

- `hindi_banking_loan.wav` — Hindi customer asking about home loan eligibility
- `hindi_insurance_claim.wav` — Hindi customer asking about claim status
- `tamil_ecommerce_return.wav` — Tamil customer wanting to return a product
- `bengali_telecom_bill.wav` — Bengali customer disputing mobile bill charges

## How to Create Samples

1. Record native speakers reading the scenario scripts
2. Or use your TTS engine to generate synthetic samples
3. Save as 16kHz mono WAV files
4. Place in this directory

## Tips

- Keep samples 15-30 seconds long
- Include some emotional variation (frustration, confusion, politeness)
- Have the speaker include specific details (amounts, dates, order numbers) for entity extraction demo
